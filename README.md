# Works
Console chess game in C using 2D arrays — supports algebraic notation input like e2 e4 
## Features
- 8x8 chess board display
- Move input like e2 e4
- Supports all pieces (King, Queen, Rook, Bishop, Knight, Pawn)

## How to Run
gcc chess.c -o chess
./chess

## How to Play
- Type move like: e2 e4
- Type q to quit
